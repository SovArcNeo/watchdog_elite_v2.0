# watchdog_elite_v2.0
A standalone, AI-powered security platform with a custom 'Matrix' GUI. Features a pure NumPy neural network for live training and anomaly detection. Includes real-time system monitoring, 3D visualizations, and threat heatmaps. A self-contained AI 'foundry' and sentinel built entirely in Python.
